TNM staging is currently the more common tumor staging method in the world. It describes the state of tumor development through three aspects: tumor primary tumor (T), regional lymph node involvement (N), and tumor metastasis.

The tumor metastasis data of THCA data has a large amount of censored data, so the regional lymph node involvement of the data volume is analyzed to find the relevant gene panel.The regional lymph node is significant related to the prognosis of THCA.

In this section we try to combine the clinical information to the risk score in three subtypes of THCA(TallCell, Follicular and Classical), the results shows that age and origional lesion together with risk score can be used to predict the lymph node involvement in THCA.


screening for gene panels assocaited with regional lymph node involvement in the classical subtype

In statistics and machine learning，the ElasticNet algorithm is a regression analysis method that performs feature selection and regularization (mathematics) simultaneously. The L1 and L2 norms are used as linear regression models for a priori regular training.

Methods:

In the Classical subtype, samples were randomly divided into a 70% training set and a 30% validation set. Pre-screening was performed using one-way logistic regression based on differentially expressed genes with a screening criterion of p < 0.1.

The ElasticNet algorithm was used to select the gene panel associated with lymph node involvement. The sample risk index was calculated based on the selected gene panel, and the ROC curve was drawn.

Conclusion:Through this result, we can find that the selected gene panel alone can be used to judge the accuracy of lymph node involvement.
